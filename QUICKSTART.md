# QUICKSTART - 3 Simple Steps

## 1. Install Packages
Double-click: **`SETUP.bat`**

Or manually:
```powershell
python -m pip install pandas numpy matplotlib seaborn scikit-learn plotly dash dash-bootstrap-components nltk textblob wordcloud scipy openpyxl
```

## 2. Run Analysis
```powershell
python run_analysis.py
```

## 3. Get Results
All outputs in **`outputs/`** folder:
- `outputs/visualizations/` - 13 PNG charts
- `outputs/reports/` - 11 CSV + 1 TXT summary
- `outputs/data/` - Complete dataset
